🎮 ViceVault

ViceVault is a modern web-based game library where users can explore, discover, and browse video games in a clean and organized interface. The platform acts as a digital vault for games, allowing users to quickly find information about their favorite titles.




🚀 Features

🎮 Game Library – Browse a large collection of games.

🔍 Search Functionality – Quickly find games by name.

🗂 Organized Categories – Games grouped for easy discovery.

⭐ Game Details – View descriptions, ratings, and release information.

⚡ Fast Performance – Optimized loading and smooth browsing experience.

  
🛠 Tech Stack

Frontend: HTML, CSS, JavaScript

API: RAWG Video Games Database API

Styling: Modern UI design

Hosting: https://vicevault.vercel.app/
